# 📚 ربات کتاب تلگرام

ربات ارسال خودکار PDF به کاربران کانال تلگرام.

---

## ⚙️ راه‌اندازی

### ۱. ساخت ربات در تلگرام
- به [@BotFather](https://t.me/BotFather) برو
- `/newbot` بزن و اسم و یوزرنیم ربات رو وارد کن
- **توکن** رو کپی کن

### ۲. گرفتن آیدی عددی خودت
- به [@userinfobot](https://t.me/userinfobot) پیام بده
- عدد `Id` رو کپی کن (این آیدی ادمین هست)

### ۳. نصب
```bash
pip install -r requirements.txt
```

### ۴. تنظیم متغیرها
```bash
# لینوکس/مک
export BOT_TOKEN="توکن_ربات_اینجا"
export ADMIN_ID="آیدی_عددی_اینجا"

# ویندوز
set BOT_TOKEN=توکن_ربات_اینجا
set ADMIN_ID=آیدی_عددی_اینجا
```

### ۵. اجرا
```bash
python bot.py
```

---

## 🚀 هاست رایگان روی Railway

1. اکانت بساز روی [railway.app](https://railway.app)
2. **New Project → Deploy from GitHub** رو بزن
3. فایل‌ها رو آپلود کن
4. تو بخش **Variables** این دو تا رو اضافه کن:
   - `BOT_TOKEN` = توکن ربات
   - `ADMIN_ID` = آیدی عددی خودت
5. **Deploy** کن — تمام!

## 📖 نحوه استفاده (ادمین)

| کار | روش |
|-----|-----|
| اضافه کردن کتاب | PDF بفرست → عنوان بنویس → توضیح بنویس (یا /skip) |
| گرفتن لینک | بعد از آپلود، لینک دیپ‌لینک خودکار ارسال می‌شه |
| لیست کتاب‌ها | `/list` |
| حذف کتاب | `/delete` |

## 🔗 فرمت لینک برای کانال

```
https://t.me/YourBot?start=book_0001
```

این لینک رو تو پست کانال بذار. کاربر کلیک کنه → وارد ربات می‌شه → PDF دریافت می‌کنه.

## 📁 ساختار فایل‌ها

```
telegram-book-bot/
├── bot.py           # کد اصلی ربات
├── requirements.txt # کتابخونه‌ها
├── README.md        # این فایل
└── books.json       # ذخیره‌سازی (خودکار ساخته می‌شه)
```

> **نکته:** فایل `books.json` خودکار ساخته می‌شه و اطلاعات کتاب‌ها رو نگه می‌داره.
> روی Railway این فایل بعد از ری‌استارت پاک می‌شه — اگه ربات رو ری‌استارت کردی دوباره PDF بفرست.
